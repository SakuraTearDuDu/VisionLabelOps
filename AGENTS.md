# AGENTS

This repository is intended to be worked on by multiple agents and humans at once.

## Operating rules

- Do not overwrite or revert other contributors' changes.
- Keep edits within your assigned scope.
- Prefer incremental, reviewable updates.
- Preserve the clean-room framing for third-party references.

## Current scope

The initial repository focus is a lightweight Python toolkit for:

- dataset audit
- format conversion
- statistics
- splits
- previews
- reports

## Guidance for agents

- Read the surrounding docs before making changes.
- Use the repository's existing package and test names when drafting examples.
- Keep Windows paths explicit in examples.
- Document assumptions instead of hiding them in code or comments.

This file is intentionally brief so it can serve as a shared coordination point without becoming stale quickly.
