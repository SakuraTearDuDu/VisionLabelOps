# Code of Conduct

VisionLabelOps is maintained as a small, engineering-focused open-source project.

## Expected behavior

- Be respectful and specific.
- Critique code and ideas, not people.
- Prefer reproducible reports with commands, file paths, and expected behavior.
- Assume good intent, but be direct about technical issues.

## Unacceptable behavior

- Harassment, threats, discrimination, or sustained hostility.
- Publishing private information without permission.
- Repeated low-signal spam in issues, discussions, or pull requests.

## Enforcement

Project maintainers may edit, remove, lock, or reject content that harms the collaboration environment or is clearly out of scope for the repository.

If you need to report conduct issues privately, contact `linzaixinroyleo@foxmail.com`.

## Scope

This applies to repository issues, pull requests, discussions, and other project-managed communication channels.
